LazyYouTubeWatcher

Set your Telegram Chat ID so the extension can send awakeness checks to you.

How to get your Telegram Chat ID
  1. 1
    Open Telegram and search for the bot named Lazy Youtube Watcher.
  2. 2
    Send the command /get_chat_id to the bot. The bot will reply with your numeric chat ID.
  3. 3
    Copy the chat ID and paste it below into the extension settings.

Tip: If the bot doesn't respond, make sure you started a conversation with it (press Start) or check that the bot username is correct.

We store this in your browser extension settings only.
